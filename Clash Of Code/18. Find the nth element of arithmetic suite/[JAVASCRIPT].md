# Goal

Find the nth element of an arithmetique suite

# Input
S: start of the arithmetique suite
D: Incrementation of the suite
N: The nth element we are looking for

# Output
The value of the element

Input:
5
4
6

Output:
25

## Solution

Character: 34
```Javascript
r=readline;print(+r()+r()*(r()-1))
```
